dir615_pppoe_atk
================

README
======
This is a script that will fetch the PPPoE username and password from the modded DIR615 by TM Malaysia for Unifi HSBB @ Malaysia.

REQUIREMENTS
============
1. bash 
2. curl
3. python

EXECUTION
=========
1. Open your terminal
2. Type "bash dir615_pppoe_atk.sh" and press Enter
3. Voila!
